חסימת רענון דף, חזרה אחורה ויציאה מהמבחן

שים לב: כדי לבדוק את המערכת המאובטחת, יש להפעיל את שרת React ולנווט לקומפוננט SecureTestDemo. במהלך המבחן לא תוכל להשתמש בפעולות העתקה, קליק ימני, או כלי פיתוח.

📋 הצג הוראות הפעלה

הוראות הפעלה:

  1. פתח טרמינל בתיקיית הפרויקט
  2. הפעל את השרת: cd client && npm start
  3. פתח דפדפן ונווט ל: http://localhost:3000
  4. ייבא את SecureTestDemo לקומפוננט הראשי
  5. לחץ על "התחל מבחן הדגמה מאובטח"
  6. בדוק את כל תכונות האבטחה:
    • נסה Ctrl+C, Ctrl+V, Ctrl+A - אמור להיחסם
    • נסה קליק ימני - אמור להיחסם
    • נסה F12 או Ctrl+Shift+I - אמור להיחסם
    • בדוק שהשאלות והתשובות מופיעות בעמודים נפרדים
    • וודא שהזיכרון מתנקה בין השלבים
