1. __At a glance__: User sees AI button color and icon indicating current status
2. __On hover__: Tooltip shows detailed status and available actions count
3. __On click__: Opens comprehensive menu with status summary, processing details, and available actions
4. __Action selection__: Direct access to transcript, summary, or test content

- __Immediate Status Visibility__: No need to click to see AI status

- __Better UX__: Status + actions in one intuitive location
- __Visual Hierarchy__: Button prominence matches AI content importance
